AI vulnerability Report 01

Amazon Q Flaw Exposes Cloud Credentials

Researchers disclosed a high-severity vulnerability in Amazon Q Developer, tracked as CVE-2026-12957, that allowed malicious repositories to execute code through crafted Model Context Protocol (MCP) configuration files. When developers opened and trusted a compromised workspace, Amazon Q automatically launched attacker-controlled MCP servers inheriting AWS credentials, API tokens, SSH agent sockets, and other environment secrets. The flaw affected plugins for Visual Studio Code, JetBrains, Eclipse, and Visual Studio. Amazon patched the issue by requiring explicit approval before launching untrusted MCP servers and released updated plugin versions. Organizations should update immediately and carefully review repositories before trusting new workspaces.

cyber espionage Report 02

Russian Smishing Campaign Targets Officials

Ukraine's Security Service (SSU), working with the FBI, uncovered a long-running Russian intelligence campaign using fake technical support SMS messages to compromise messaging accounts. The operation targeted government officials, military personnel, politicians, and activists across Ukraine, Europe, and the United States. Victims received convincing messages impersonating trusted support services, directing them to phishing pages that harvested authentication credentials for secure messaging platforms. Investigators linked the activity to Russian intelligence services conducting cyber espionage aimed at collecting sensitive military, political, and economic information. Security agencies recommend verifying support communications, enabling multifactor authentication, and remaining vigilant against phishing attempts.

malware loader Report 03

SharkLoader Deploys Cobalt Strike

Researchers identified a new malware loader named SharkLoader used in the StrikeShark campaign to deploy Cobalt Strike Beacon across compromised systems. The campaign targets diplomatic organizations, government agencies, and software development companies in multiple countries, including Indonesia, Taiwan, and Colombia. Attackers gain initial access by exploiting publicly known vulnerabilities in products such as Microsoft Exchange, Fortinet FortiOS, F5 BIG-IP, Cisco IOS XE, and Openfire. After compromise, SharkLoader uses DLL sideloading, persistence techniques, and open-source reconnaissance tools to establish long-term access before deploying Cobalt Strike for post-exploitation activities. Researchers believe the operation is linked to a Chinese-speaking threat actor.
