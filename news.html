supplychain malware Report 01

PyPI packages spread ZiChatbot malware.

Researchers uncovered malicious Python packages on PyPI distributing the ZIChatBot malware through software supply chain abuse. The packages impersonated legitimate developer tools, tricking users into downloading infected dependencies during development workflows. Once executed, the malware established persistence, collected system information, and enabled remote command execution. This campaign highlights how attackers increasingly target open-source ecosystems to compromise developers and downstream environments. The incident reinforces the importance of dependency verification, package reputation analysis, isolated development environments, and continuous monitoring of third-party libraries. Organizations relying heavily on open-source packages face elevated risks from increasingly sophisticated repository-based malware distribution campaigns.

Nvidia GB300 dubbed ultimate AI computer.

NVIDIA’s GB300 AI system is drawing major industry attention after Elon Musk described it as the “best AI computer.” Built on the Blackwell Ultra architecture, the platform delivers massive computational power designed for advanced reasoning models, multimodal AI, and large-scale inference workloads. botnet malware Report 03

Mirai botnet exploits exposed ADB services.

Researchers identified a Mirai-based botnet called xLabsV1 exploiting exposed Android Debug Bridge services to compromise internet-connected devices. Attackers scan for improperly secured ADB interfaces, gain unauthorized access, and deploy malware to recruit devices into the botnet. Once infected, systems can be used for distributed denial-of-service attacks, malicious proxying, and coordinated network abuse. The campaign demonstrates how insecure remote management interfaces continue enabling large-scale compromise operations. Organizations and users should disable unnecessary ADB exposure, enforce authentication controls, and isolate internet-facing devices to reduce risks associated with automated botnet propagation and exploitation targeting weakly secured Android environments.
